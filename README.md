# XAU/USD AI Trading Model

## Multi-Timeframe Gold Trading AI

AI-powered trading model trained on XAU/USD market data using multi-timeframe analysis:

- 1 Minute (M1)
- 5 Minute (M5)
- 15 Minute (M15)

The system uses machine learning and technical analysis to identify trading opportunities, trend direction, and momentum behavior in gold markets.
